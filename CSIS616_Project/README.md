       For windows you can type:
       
        main <args>
        
       or
       
        main
        
       For Linux/UNIX you can type:
       
        ./main <args>
        
       or
       
        ./main
        
       Such that the args is a txt file that contains an expression.
       
You can enter no txt file and it will prompt you to enter expression with keyboard.
